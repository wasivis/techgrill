💻 TechGrill: LLM-Powered Technical Interviewer

⌨️ Tech Stack: Next.js 15, Groq (Llama 3.3), Neon Postgres (Drizzle ORM), Tailwind CSS.

🚀 Key Features:

  🔷 State-Machine Conversational Flow: Implemented custom system prompts to manage interview phases and exit signals.

  🔷 Resilient Rate Limiting: Developed a "Fail-Open" token bucket strategy to handle high-traffic scenarios.

  🔷 Serverless Architecture: Utilized Next.js Server Actions and Neon's serverless driver for low-latency database operations.

  🔷 Atomic Evaluation: Built a post-interview grading engine that analyzes code quality and communication style.
